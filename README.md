# Study Quotes App

A simple Android app that displays motivational quotes to inspire your day.

---

## 🚀 Features

* 📖 Displays motivational quotes
* 🔄 Button to generate a new quote
* ⚡ Lightweight and fast
* 📱 Built using Java in Android Studio

---

## 🛠️ Tech Stack

* Java
* Android SDK
* XML (UI Design)
* use of groovy gradle built (not kothin)

---

##  How to run 

1. Open the app
2. A motivational quote is displayed
3. Tap **"New Quote"** to see another quote

---

##  Project Structure

* `MainActivity.java` → App logic
* `activity_main.xml` → UI layout
* `strings.xml` → App name and text
